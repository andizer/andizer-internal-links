Andizer Internal Links retrieves internal link data stored by Yoast SEO, helping you gather incoming
and outgoing links for any post.

== Description ==

**Andizer Internal Links** is a plugin that extends the functionality of the popular Yoast SEO plugin by analyzing and
retrieving internal link data. It provides useful insights into your internal linking structure, which helps you optimize
your content for SEO.

With this plugin, you can:

- Identify possible **unused internal links** across your site.
- View **incoming links** for any specific post.
- View **outgoing links** for any specific post.
- Enhance your site's SEO by improving your internal linking strategy based on the data collected by Yoast SEO.

The plugin is particularly useful for site administrators, content creators, and SEO professionals who want to monitor and
optimize internal links within their content.

== Installation ==

1. Upload the `andizer-internal-links` directory to the `/wp-content/plugins/` directory, or install the plugin through
the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. The plugin will automatically retrieve internal link data stored by the Yoast SEO plugin.

== Frequently Asked Questions ==

= Does this plugin work without Yoast SEO? =

No, this plugin relies on the internal linking data stored by the Yoast SEO plugin. You need to have Yoast SEO installed
and activated for this plugin to function.
